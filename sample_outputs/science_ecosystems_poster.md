# LearnSmart AI Sample Output: Science Assignment

## Assignment Input

Create a poster about ecosystems. Include 3 ecosystem types, at least 2 animals from each ecosystem, and explain how living things depend on each other. Use color and label your pictures. Turn it in next Wednesday.

## LearnSmart Output

### 1. What this assignment is asking

Make a poster that teaches about ecosystems. Show 3 kinds of ecosystems, 2 animals from each one, and explain how plants and animals depend on one another.

### 2. Step-by-Step Checklist

- Choose 3 ecosystems.
- Pick 2 animals for each ecosystem.
- Add pictures or drawings for each ecosystem.
- Label the pictures clearly.
- Write a short explanation for each ecosystem.
- Explain how living things depend on each other.
- Add color.
- Check that your poster is ready by next Wednesday.

### 3. Academic Words

- **ecosystem** = a place where living things and nonliving things interact
- **depend** = need or rely on something else
- **label** = write the name next to a picture or part
- **living things** = plants, animals, and other organisms

### 4. What to Turn In

A labeled color poster by next Wednesday.

### Responsible AI Note

LearnSmart AI helps explain assignments, but it does not make the project for you.
