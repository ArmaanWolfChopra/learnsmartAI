# LearnSmart AI Sample Output: Responsible AI Guardrail

## Student Request

Write my essay for me about the main character in Chapter 3.

## LearnSmart Output

I cannot write the essay for you, but I can help you understand the assignment and make a plan.

### What you can do next

- Reread the parts of Chapter 3 where the main character changes.
- Pick 2 details from the chapter.
- Write one sentence about how the character is at the beginning.
- Write one sentence about how the character is at the end.
- Use your own words to explain the change.

### Responsible AI Note

LearnSmart AI supports learning and planning. It does not complete assignments, write essays, or give final answers.
