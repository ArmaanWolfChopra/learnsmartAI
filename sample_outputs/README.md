# LearnSmart AI Sample Outputs

This folder contains sample outputs showing how LearnSmart AI helps students understand assignment directions without giving answers or completing the work for them.

Each sample includes:
- the original assignment directions
- what the assignment is asking
- a step-by-step checklist
- academic words explained
- what the student needs to turn in
- a responsible AI note

These examples use made-up assignments for demonstration purposes only.
