# Responsible AI Design

LearnSmart AI was designed to support student understanding, not replace student effort.

## Allowed Help

The app can:

* clarify confusing directions
* explain academic vocabulary
* organize tasks into steps
* identify what needs to be submitted
* help students plan how to begin

## Not Allowed

The app should not:

* write final answers
* complete essays
* solve homework problems
* create full submissions
* replace teacher instructions

## Example Guardrail

If a student asks, "Write this essay for me," LearnSmart AI should respond by saying that it cannot write the essay, but it can help the student understand the prompt, make a plan, and identify the first step.

## Cybersecurity and Privacy

The app is designed to avoid collecting unnecessary personal information. API keys, credentials, and private configuration files are not stored in the public repository.
